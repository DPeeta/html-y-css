html

HTML5 es la última versión del lenguaje de marcado HTML (HyperText Markup Language). Es un estándar utilizado para estructurar y presentar el contenido de las páginas web en Internet.

Etiquetas

Existen dos tipos de etiquetas

1  De apertura y cierre {" "} (normalmente encierran contenido)
<a> hola </a>

2  De cierre automático{" "} (no llevan contenido pero si atributos los cuales se separan por un espacio )
 <img url=”img.jpg” />   {" "} la diagonal del final no es obligatoria

Documentación oficial de html5 de Mozilla
https://developer.mozilla.org/es/docs/orphaned/Web/HTML/Inline_elements

Estructura de las etiquetas:
  <nombre de la etiqueta  atrib1:”val”  atrib2:”val”>  
           Contenido
</nombre de la etiqueta>  

Estructura base de HTML5:
<!DOCTYPE html> // indica al navegador q s está usando HTML5
<html lang="en">   // etiqueta raíz -lang define el lenguaje
<head> // aquí van los metadatos (recursos informativos o de utilidad -nombre -descripción -íconos etc)
<meta charset="UTF-8"> // permite agregar caracteres especiales -ñ -á etc
<meta name="viewport" content="width=device-width, initial-scale=1.0"> // para adaptarla a dispositivos móviles
<title>Document</title> // título de la pág q se muestra en el nav
<link rel="stylesheet" href="./prueba.css">
</head>
<body>
</body>
</html>

<!-- así se hace un comentario en HTML, lo que esté contenido aquí no aparecerá en el navegador{" "} {" "} -->

Etiquetas de Textos

Etiquetas heading o de encabezado {" "} (generalmente son títulos y determinan jerarquía )
solo debe haber 1 etiqueta H1 por documento y esta representa el título principal de nuestro documento
<h1> Título 1 </h1>
<h2> Título 2 </h2>
<h3> Título 3 </h3>
<h4> Título 4 </h4>
<h5> Título 5 </h5>
<h6> Título 6 </h6>

Etiquetas de estilo {" "} (con y sin valor semántico)

<hr> // representa un cambio de tema y visualmente genera una línea vertical
<br> // salto de línea

  <em> itálica </em> // tiene valor semántico
<i> itálica </i> // solo visual

<strong> strong </strong>  // tiene valor semántico
<b> strong </b> // solo visual

<u>subrayado</u>
<small>más pequeño</small>
<mark>resaltado</mark>  // tiene valor semántico
<s>lo tacha</s>

A <sup>superíndice</sup>
A <sub>subíndice</sub>
