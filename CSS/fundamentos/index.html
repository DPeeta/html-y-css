Cascada CSS

Css se basa en dos reglas para determinar que estilos se van a aplicar, la cascada y la especificidad .

Especificidad(método principal de CSS) tiene mas importancia que la cascada: es el valor que se le da a los selectores en CSS

cascada: siempre se aplicarán los últimos estilos en ser declarados, tanto a nivel de selectores como a nivel de propiedades, pero solo si se tienen especificidades iguales

!important y los estilos en linea son mala práctica porque rompen la especificidad

La herencia es reversible. Si doy una propiedad a todos los elementos del body todos la tomarán a excepción de los elementos específicos a los que les de otro valor

no todas las propiedades de css se heredan (borde, padding, width, height )

inherit obliga al elemento a heredar ese valor de su padre
