Unidades de medida

Establecer tamaños de letra relativos permite webs mas adaptables RECOMENDADO(rem)
em es una unidad de medida relativa al tamaño fuente del padre o del ancestro mas cercano con un font-size definido.
rem es una unidad relativa al root del html (la etiqueta html)

El viewport es el área visible del navegador
vw es una unidad de medida relativa al ancho del viewport (puede tomar valores del 0 al 100)
vh es una unidad relativa al alto del viewport
El porcentaje es relativo al tamaño del padre del elemento, normalmente el ancho de una página si está establecido pero el alto no, pr lo tanto si se quiere establecer un valor de height en porciento deberá especificarse el valor de height del padre

el ancho de la imagen se puede establecer en px, cm, mm o pt(puntos)
