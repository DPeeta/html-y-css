Colapso de márgenes

cuando existe un elemento debajo del otro, la zona que los separa, asume el margen mayor entre de ambos , no la suma de estos.


el colapso de márgenes también afecta a elementos hijos y padres respecto a los margenes superiores(en caso del 1er elemento) , o inferiores (en caso del último elemento)

Si quiero mantener ambos márgenes debo establecer un border o padding entre estos dos elementos para delimitarlos
