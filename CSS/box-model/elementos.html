Soy un elemento bloque

para saber si un elemento es bloque o línea podemos inspeccionar y ver que valor tiene en la propiedad display desde Styles o desde Computed. Además de eso, podemos diferenciarlos porque los elementos en bloque generan un salto de línea

Si el elemento no es bloque no se podrán aplicar las propiedades width y height, y el margen o borde se sobrepondrá a otros elementos cercanos

Si establecemos la propiedad display:inline-block podemos aplicar correctamente los estilos y los elementos seguirán posicionándose a continuación y no en una nueva línea
