Mediaquery

Mediaquery es un tipo de condicional en css que solo se va a aplicar cuando se cumpla x condición, lo que permite crear diseños adaptables a cualquier dispositivo

max-width indica el valor máximo que debe tener el viewport para que se apliquen los estilos dentro de un mediaquery , de igual forma con min-width y los valores mínimos

Se define debajo de todo el css de la siguiente forma @media(min-width: 600px){}

para max-width los mediaquery deben estar ordenados de mayor a menor y para min-width de menor a mayor
