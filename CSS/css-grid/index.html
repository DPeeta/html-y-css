si los elementos dentro del grid no tienen un alto definido, al declarar display: grid; estos se estirarán para ocupar todo el display
grid-template-columns: repeat(6, 1fr); definirá el numero de columnas y grid-template-rows: repeat(3, 1fr); el número de filas.
En los grid-items podemos definir la posición en el grid
grid-column: 1/5; define que columnas ocupará y
grid-row: 1/2; define que filas ocupará.
gap: 20px; permite definir un espaciado entre elementos
