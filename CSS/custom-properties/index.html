Custom Properties (propiedades personalizadas) o variables de css

Las custom properties pueden almacenar cualquier valor permitido en las propiedades de CSS
Esto nos sirve para modificar de forma masiva en lugar que tener que hacerlo valor por valor

Para definir los nombres de las custom properties declaramos un selector (pseudo-clase) para darles más especificidad y que sea más difícil renombrar.
El selector :root{} representa a la etiqueta html
para declararla debemos escribir --NombreCualquieraQRepresente: valor;
y ya de aquí en adelante podremos utilizar esta variable (llamándola con la función var) para acceder a su valor
ejemplo:
:root{ --color-primario: green ; }
.title{ color: var(--color-primario); }
