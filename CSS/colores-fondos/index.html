Colores

Al pasar el mouse sobre un color en css este muestra su código RGB. y si hacemos click en la barra superior estaremos alternando entre las diferentes representaciones
Los colores pueden representarse por:

  1. palabra clave

    2. aquí puedes ver la palabra clave que representa un color https://htmlcolorcodes.com/

  2. valores rgb

    3. rgb admite valores en cada posición entre 0-255.
    Se define con 3 valores, color: rgb(red, green, blue);
    y también podemos añadir un 4to valor del 0 al 1 que define la transparencia, siendo 1 sólido y 0 totalmente transparente

  3. hexadecimal

    4. Se representa por el símbolo # seguido de 6 valores, los dso 1ros representan al rojo, los dos centrales al verde y los dos últimos al azul, igual que el código rgb. Los valores van del 0 a la letra f. Se pueden añadir dos valores adicionales al final que también representarán la transparencia

Gradiente de fondo

podemos usar el valor linear gradient en background-image, así: background-image: linear-gradient(purple, white);
y pasarle dos colores para crear el efecto gradiente

Imagen de fondo con gradiente de colores

también podemos definir los valores de linear-gradient con colores transparentes, y separado por una coma establecer la imagen para lograr este efecto
background-image:linear-gradient(rgba(0, 128, 0, 0.58), rgba(255, 0, 255, 0.571)), url('./logo\ PMS.png');
