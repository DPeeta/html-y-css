Flexbox es el módulo de caja flexible de css, creado para diseñar layouts, trabaja de manera unidimensional (con un eje principal), los elementos dentro de este se vuelven flexibles y podemos controlar a voluntad su comportamiento (crecimiento, alineación o distribuir el espacio disponible)
esto ayuda a desarrollar RESPONSIVE
se declara(display: flex;) Al establecer esto lo elementos se posicionan de forma horizontal y no vertical, si no definimos ancho este varía de acuerdo a su contenido, y si no definimos alto, este ocupará todo el espacio posible dentro de la flex box

flex-direction: row; determina el eje principal
valores posibles (row, row-reverse, column, column-reverse ) Main-axis : eje principal
Cross-axis : eje secundario
